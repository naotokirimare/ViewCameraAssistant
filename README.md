- キャッシュ対策を `?v=60` に更新

## 同期モードの使い方

1. MacBookでこのフォルダを開く
2. ターミナルで以下を実行

```bash
node sync-server.js
```

3. MacのIPアドレスを確認する
   - 例: `192.168.1.23`
4. iPhone / iPad / MacBookでアプリを開く
5. 測定画面 → 同期モード α → 同期サーバーURL に入力

```text
ws://192.168.1.23:8787
```

6. 各端末で「同期開始」を押す
7. iPhone側だけ「測定開始」を押してセンサーを取得する

## 注意

- iPhoneのブラウザ単体では同一Wi-Fi内の同期サーバーになれないため、MacBookなどで `sync-server.js` を起動します。
- センサー取得はiPhone側で行います。
- iPad / MacBook側は同じUIを表示し、ゼロ補正・基準保存・反映などを遠隔操作できます。
- HTTPS環境では `ws://` がブロックされる場合があります。その場合はローカルHTTPで開いてください。
